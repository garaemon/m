# syntax highlight samples

## python syntax highlight

```python
import foo
import bar

def main():
    print('Hello World')

if __name__ == '__main__':
    main()
```

## c++ syntax highlight

```c++
#include <iostream>

int main() {
    std::cout << "Hello World" << std::endl;
    return 0;
}
```

## c syntax highlight

```c
#include <stddio.h>

int main() {
    printf("Hello World\n");
    return 0;
}
```

## bash syntax highlight

```bash
#!/bin/bash

echo Hello World
```
